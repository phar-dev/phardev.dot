# Shadcn/UI Documentation

This folder contains local documentation and references for using shadcn/ui components in the SGC React project.

## Official Documentation

- [Shadcn/UI Official Site](https://ui.shadcn.com/)
- [Installation Guide](https://ui.shadcn.com/docs/installation)
- [Components Documentation](https://ui.shadcn.com/docs/components)

## Project-Specific Notes

- Components are installed in `resources/js/components/ui/`
- Use `pnpm dlx shadcn@latest add <component>` to add new components
- Follow project conventions from `tailwind-4` and `react-19` skills
- Customize components by editing the source files directly

## Common Components

- Button: Basic interactive button with variants
- Input: Form input fields
- Card: Container for content
- Dialog: Modal dialogs
- Table: Data tables

For more details, refer to the official documentation above.
